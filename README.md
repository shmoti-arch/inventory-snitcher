# Inventory Snitcher
### by shmoti <3

- basically a sh script for roblox inventory look up
- can access player's models and their ids
- uses `curl`

## installation ⚡︎

```bash
git clone https://github.com/shmoti-arch/inventory-snitcher
cd inventory-snitcher
chmod +x script.sh
./script.sh
```
